Constant Intensity

Given a set of age nodes \(0=x_0,x_1,…,x_n\) and migration probabilities \(\migprob_0, \migprob_1,\ldots,\migprob_{n-1}\) for age intervals \([x_i,x_{i+1})\), the instantaneous migration intensity over interval \([x_i,x_{i+1})\) is constant and equal to \begin{equation} \intensity_i=\frac{-1}{x_{i+1}-x_i}\log(1-\migprob_i) \end{equation}
