ModalDeathAge

ModalDeathAge(MortHandle,[AgeLo],[AgeHi],[Tol])

Returns the modal age of death between ages AgeLo and AgeHi to tolerance Tol for mortality object MortHandle. If AgeLo is not specified it defaults to 15. If AgeHi is not specified it defaults to 110. If Tol is not specified it defaults to 0.0001.
