SurvProb

SurvProb(MortHandle,Age,[AgeInterval])

Returns the survival probability for age interval [Age,Age+AgeInterval). If AgeInterval is not specified it defaults to 1.0. The survival probability \(_n p_x\) is one minus the death probability $$ _n p_x = 1- {_n q_x} $$
