CreateResult

CreateResult(Handle,DataRange,[ResultsType])

Creates a results object from a datarange. Handle is the name of the results object. DataRange is the range of data which will be placed in a subtable named ResultsGrid. The optional parameter ResultsType specifies the value of the ResultsType body property. If not specified it defaults to RESULT. The object created by this function will also have a body property ResultSize which gives the rows and columns in the ResultsGrid subtable.
