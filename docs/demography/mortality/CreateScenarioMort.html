CreateScenarioMort

CreateScenarioMort(NewMortHandle,MortHandle,ScenarioHandle)

Creates a new mortality object by applying a scenario to an existing mortality. NewMortHandle is the name of the new mortality object. MortHandle is the name of the original mortality object. ScenarioHandle is the name of a results object which defines the scenario. It must have a body property Scenario the value of which defines the type of scenario that will be applied.

MULTIDEC Scenario

For a ScenarioHandle with Scenario property MULTIDEC a new mortality object will be created corresponding to a subset of causes of death. MortHandle must be a multiple decrement mortality object. ScenarioHandle must have a body property Cause which is a comma-separated list of cause codes to be applied.
