SurvFrac

\( \newcommand{\sfrac}{l} \newcommand{\force}{\mu} \newcommand{\cause}{\alpha} \)

SurvFrac(MortHandle,[Age],[Cause])

Returns the survival fraction for the supplied age. If [Age] is not specified then the function returns all age nodes, survival fractions and years lived used to build the curve. The survival fraction \(l(x)\) at Age \(x\) is related to the force of mortality \(\mu(x)\) through the expression $$ \sfrac(x) = \exp\left(-\int_0^x \force(a)da\right) $$ If Cause is specified then this function returns the cause-specific survival fraction \begin{equation} \sfrac_\cause(x)=\int_x^\infty \sfrac(a)\force_\cause(a) da \end{equation} where Cause is a comma-separated list of cause codes, possibly prefixed by '~' to denote the complement.
