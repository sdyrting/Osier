ListToMatrix

ListToMatrix(RowLabels,ColumnLabels,Values,[LabelType])

Constructs a matrix from lists of row labels, column labels and values. The inverse of MatrixToList. RowLabels is a vector of row label values, ColumnLabels is a vector of column label values, and Values is a vector of matrix elements. The matrix elements are arranged in increasing row and column labels. The optional parameter LabelType specifies how the labels are interpretted. It can take values: 'NUMBER', 'DATE', 'STRING' or 'INTERVAL'. If not specified then it defaults to 'NUMBER'.

Example: ListToMatrix(RowLablels,ColumnLabels,Values,"DATE")
